# AuditIA 🚀

Tu **AI Product Manager** para análisis automático de UX, Performance, Accesibilidad y SEO.

## 🎯 Idea

AuditIA es una plataforma donde introduces una URL y la IA actúa como un "AI Product Manager". Analiza automáticamente la web en términos de:

- **UX** - Experiencia de usuario
- **Performance** - Velocidad y optimización
- **Accesibilidad** - WCAG compliance y usabilidad
- **SEO** - Posicionamiento en buscadores

En lugar de solo dar métricas, **traduce los problemas en decisiones de producto claras**.

### Características Principales

✨ **Análisis Inteligente** - La IA genera un roadmap visual priorizado con tareas concretas

🎯 **Priorización Inteligente** - Diferencia entre quick wins y high impact

📋 **Proyectos Iterativos** - Cada análisis se convierte en un proyecto

📊 **Seguimiento de Impacto** - Marca tareas como completadas y re-analiza para medir mejoras

🔄 **Mejora Continua** - Vuelve a analizar la web para medir el impacto real de las mejoras

## 🚀 Inicio Rápido

### Requisitos Previos
- Node.js 18+ 
- pnpm (recomendado)

### Instalación

```bash
# Clonar el repositorio
git clone <repository-url>
cd auditia

# Instalar dependencias
pnpm install

# Ejecutar servidor de desarrollo
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
